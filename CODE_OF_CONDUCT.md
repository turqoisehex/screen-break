# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## Reporting

If you experience or witness unacceptable behavior, please report it by opening a [GitHub issue](https://github.com/turqoisehex/screen-break/issues) with the label `conduct`, or use [GitHub's private reporting](https://github.com/turqoisehex/screen-break/security/advisories/new) if you prefer confidentiality.

## Enforcement

This is a side project maintained in spare time. Reports will be reviewed and addressed as promptly as possible. Enforcement actions may include warnings, temporary bans, or permanent bans from project spaces.
