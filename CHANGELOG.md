## [2.1.0] - 2026-03-14

### Added
- Cross-platform support (macOS and Linux builds)
- Floating widget for Windows 11 accessibility
- Multi-monitor overlay support
- Guided eye exercises (circle, figure-8, near-far focus)
- Breathing exercises (box, relaxing, energizing patterns)
- Desk exercises with animated instructions
- Hydration tracking with daily goal
- Focus sessions (Pomodoro-style with task naming)
- Daily/weekly visual reports
- Custom sounds (.wav, .mp3, .ogg)
- Custom motivational messages
- Themes (dark, light, nord)
- Presentation mode (auto-pause during fullscreen)
- Low energy mode
- Advance warning before breaks
- Mini reminders (posture, hydration, blink nudges)
- Notes capture during breaks
- Strict mode (prevent skipping, snooze only)
- Screen dimming effect
- Work hours configuration

### Fixed
- Safety valves for timer edge cases
- TclError protection for window operations
- Idle recovery improvements
- Unicode print crash on Windows cp1252 consoles
